# เริ่มต้นใช้งาน Cloud ด่วน

NetRecon Cloud เป็นวิธีที่เร็วที่สุดในการเริ่มต้น ไม่ต้องตั้งค่าเซิร์ฟเวอร์ ไม่ต้องใช้ Docker เพียงสมัครสมาชิก ติดตั้ง probe และเริ่มค้นหาเครือข่ายของคุณ

## ขั้นตอนที่ 1: สร้างบัญชีของคุณ

1. ไปที่ [app.netreconapp.com](https://app.netreconapp.com) แล้วคลิก **สมัครสมาชิก**
2. กรอกอีเมล ชื่อบริษัท และรหัสผ่าน
3. ยืนยันที่อยู่อีเมลของคุณ
4. เข้าสู่ระบบ NetRecon Dashboard

## ขั้นตอนที่ 2: เพิ่มไซต์แรกของคุณ

1. ใน Dashboard ไปที่ **ไซต์** ในแถบด้านข้าง
2. คลิก **เพิ่มไซต์**
3. กรอกชื่อและที่อยู่ของไซต์ (เช่น "สำนักงานใหญ่ — กรุงเทพฯ")
4. บันทึกไซต์

## ขั้นตอนที่ 3: ติดตั้ง Probe

แต่ละไซต์ต้องมี probe อย่างน้อยหนึ่งตัวสำหรับการค้นหาและตรวจสอบเครือข่าย

### ตัวเลือก A: NetRecon OS (แนะนำ)

1. ไปที่ **ไซต์ → [ไซต์ของคุณ] → Probes → เพิ่ม Probe**
2. เลือก **NetRecon OS** แล้วดาวน์โหลดอิมเมจสำหรับฮาร์ดแวร์ของคุณ
3. เขียนอิมเมจลงการ์ด SD หรือ SSD โดยใช้ [balenaEtcher](https://etcher.balena.io/)
4. เชื่อมต่อ probe เข้ากับเครือข่ายของคุณผ่าน Ethernet
5. เปิดเครื่อง — probe จะเชื่อมต่อกับบัญชี cloud ของคุณโดยอัตโนมัติผ่าน Cloudflare Tunnel

### ตัวเลือก B: Docker บนเซิร์ฟเวอร์ที่มีอยู่

```bash
# ดึงและรันคอนเทนเนอร์ probe
docker run -d \
  --name netrecon-probe \
  --network host \
  --restart unless-stopped \
  -e ENROLLMENT_TOKEN="your-token-from-dashboard" \
  netrecon/probe:latest
```

รับ enrollment token จาก **ไซต์ → [ไซต์ของคุณ] → Probes → เพิ่ม Probe → Docker**

### ตัวเลือก C: Virtual Machine

1. ดาวน์โหลดไฟล์ OVA จาก Dashboard
2. นำเข้าไปยัง VMware, Proxmox หรือ Hyper-V
3. กำหนดค่า VM ด้วย **bridged networking** (จำเป็นสำหรับการสแกน Layer 2)
4. เริ่มต้น VM — จะปรากฏใน Dashboard ของคุณโดยอัตโนมัติ

## ขั้นตอนที่ 4: เริ่มสแกน

เมื่อ probe ออนไลน์แล้ว:

1. ไปที่ **ไซต์ → [ไซต์ของคุณ] → อุปกรณ์**
2. คลิก **สแกนเลย** หรือรอการค้นหาอัตโนมัติ (ทำงานทุก 15 นาที)
3. อุปกรณ์ที่ค้นพบจะปรากฏในรายการอุปกรณ์

## ขั้นตอนที่ 5: ติดตั้งแอปมือถือ

ดาวน์โหลด **NetRecon Scanner** จาก Google Play Store สำหรับการสแกนเครือข่ายขณะเดินทาง:

- สแกนเครือข่ายใดก็ได้ที่โทรศัพท์ของคุณเชื่อมต่ออยู่
- ผลลัพธ์จะซิงค์โดยอัตโนมัติไปยัง cloud dashboard ของคุณ
- ดู [ภาพรวม Scanner](../scanner/overview) สำหรับรายละเอียด

## ขั้นตอนถัดไป

- **ติดตั้ง agent** บน endpoint เพื่อการมองเห็นที่ลึกขึ้น → [การติดตั้ง Agent](../agents/overview)
- **ตั้งค่าการแจ้งเตือน** สำหรับอุปกรณ์ใหม่ ช่องโหว่ หรือการหยุดทำงาน
- **กำหนดค่าการเชื่อมต่อ** กับเครื่องมือที่มีอยู่ของคุณ (LDAP, SIEM, Jira, ServiceNow)
- **เชิญทีมของคุณ** ผ่าน **ตั้งค่า → การจัดการทีม**

## Cloud เทียบกับ Self-Hosted

| ฟีเจอร์ | Cloud | Self-Hosted |
|---|---|---|
| การจัดการเซิร์ฟเวอร์ | จัดการโดย NetRecon | คุณจัดการเอง |
| ตำแหน่งข้อมูล | NetRecon Cloud (EU) | โครงสร้างพื้นฐานของคุณ |
| การอัปเดต | อัตโนมัติ | ด้วยตนเอง (docker pull) |
| Cloudflare Tunnel | รวมอยู่แล้ว | คุณกำหนดค่าเอง |
| ราคา | สมัครสมาชิก | License key |
