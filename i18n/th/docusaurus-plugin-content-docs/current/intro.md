# เริ่มต้นใช้งาน NetRecon

NetRecon เป็นแพลตฟอร์มข่าวกรองเครือข่ายที่สร้างขึ้นสำหรับ MSP และทีม IT โดยมีฟีเจอร์การค้นหาเครือข่ายอัตโนมัติ การจัดทำรายการอุปกรณ์ การสแกนช่องโหว่ การจัดการคอนฟิกูเรชัน และการตรวจสอบแบบเรียลไทม์ ทั้งหมดเข้าถึงได้ผ่านแดชบอร์ดส่วนกลาง แอปมือถือ และ REST API

## เลือกรูปแบบการติดตั้ง

<div className="row" style={{marginTop: '1.5rem'}}>

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Self-Hosted

ติดตั้ง NetRecon บนโครงสร้างพื้นฐานของคุณเองโดยใช้ Docker Compose ควบคุมข้อมูลได้อย่างสมบูรณ์ ไม่มีการพึ่งพาภายนอก

- [ข้อกำหนดระบบ](self-hosting/requirements)
- [คู่มือการติดตั้ง](self-hosting/installation)
- [เอกสารอ้างอิงคอนฟิกูเรชัน](self-hosting/configuration)

**เหมาะสำหรับ:** องค์กรที่มีข้อกำหนดด้านอธิปไตยข้อมูลที่เข้มงวด เครือข่ายแบบ air-gapped หรือมีโครงสร้างพื้นฐานเซิร์ฟเวอร์อยู่แล้ว

</div>

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Cloud (SaaS)

เริ่มต้นใช้งานทันทีกับ NetRecon Cloud ไม่ต้องตั้งค่าเซิร์ฟเวอร์ เพียงติดตั้ง probe และเริ่มสแกน

- [คู่มือเริ่มต้นใช้งานด่วน](cloud/quickstart)

**เหมาะสำหรับ:** ทีมที่ต้องการเริ่มใช้งานได้อย่างรวดเร็วโดยไม่ต้องจัดการโครงสร้างพื้นฐานเซิร์ฟเวอร์

</div>

</div>

## ส่วนประกอบของแพลตฟอร์ม

| ส่วนประกอบ | รายละเอียด |
|---|---|
| **Dashboard** | แผงควบคุมบนเว็บสำหรับฟีเจอร์ทั้งหมดของ NetRecon |
| **NetRecon Scanner** | แอป Android สำหรับสแกนเครือข่ายขณะเดินทาง ([เรียนรู้เพิ่มเติม](scanner/overview)) |
| **Admin Connect** | แอปจัดการ Android สำหรับการบริหารระยะไกล ([เรียนรู้เพิ่มเติม](admin-connect/overview)) |
| **Agents** | เอเจนต์ขนาดเบาสำหรับ endpoint Windows, macOS และ Linux ([การติดตั้ง](agents/overview)) |
| **Probes** | เซ็นเซอร์เครือข่ายแบบฮาร์ดแวร์หรือ VM สำหรับการตรวจสอบอย่างต่อเนื่อง |
| **API** | RESTful API สำหรับระบบอัตโนมัติและการเชื่อมต่อ ([เอกสารอ้างอิง API](api/overview)) |

## ต้องการความช่วยเหลือ?

- เรียกดูเอกสารโดยใช้แถบด้านข้าง
- ตรวจสอบ [เอกสารอ้างอิง API](api/overview) สำหรับรายละเอียดการเชื่อมต่อ
- ติดต่อ [support@netreconapp.com](mailto:support@netreconapp.com) เพื่อขอความช่วยเหลือ
