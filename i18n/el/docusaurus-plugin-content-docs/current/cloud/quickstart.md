# Γρήγορη Εκκίνηση Cloud

Το NetRecon Cloud είναι ο γρηγορότερος τρόπος για να ξεκινήσετε. Χωρίς εγκατάσταση διακομιστή, χωρίς Docker — απλά εγγραφείτε, αναπτύξτε έναν αισθητήρα και αρχίστε να ανακαλύπτετε το δίκτυό σας.

## Βήμα 1: Δημιουργήστε τον Λογαριασμό σας

1. Μεταβείτε στο [app.netreconapp.com](https://app.netreconapp.com) και κάντε κλικ στο **Εγγραφή**
2. Εισάγετε το email, το όνομα εταιρείας και τον κωδικό πρόσβασής σας
3. Επιβεβαιώστε τη διεύθυνση email σας
4. Συνδεθείτε στον Πίνακα Ελέγχου του NetRecon

## Βήμα 2: Προσθέστε την Πρώτη σας Τοποθεσία

1. Στον Πίνακα Ελέγχου, μεταβείτε στις **Τοποθεσίες** στην πλαϊνή μπάρα
2. Κάντε κλικ στο **Προσθήκη Τοποθεσίας**
3. Εισάγετε ένα όνομα και μια διεύθυνση για την τοποθεσία (π.χ. "Κεντρικό Γραφείο — Αθήνα")
4. Αποθηκεύστε την τοποθεσία

## Βήμα 3: Αναπτύξτε έναν Αισθητήρα

Κάθε τοποθεσία χρειάζεται τουλάχιστον έναν αισθητήρα για ανακάλυψη και παρακολούθηση δικτύου.

### Επιλογή A: NetRecon OS (Συνιστάται)

1. Μεταβείτε στο **Τοποθεσίες → [Η Τοποθεσία σας] → Αισθητήρες → Προσθήκη Αισθητήρα**
2. Επιλέξτε **NetRecon OS** και κατεβάστε την εικόνα για το hardware σας
3. Εγγράψτε την εικόνα σε κάρτα SD ή SSD χρησιμοποιώντας το [balenaEtcher](https://etcher.balena.io/)
4. Συνδέστε τον αισθητήρα στο δίκτυό σας μέσω Ethernet
5. Ενεργοποιήστε τον — ο αισθητήρας θα συνδεθεί αυτόματα στον cloud λογαριασμό σας μέσω Cloudflare Tunnel

### Επιλογή B: Docker σε Υπάρχοντα Διακομιστή

```bash
# Κατεβάστε και εκτελέστε το container του αισθητήρα
docker run -d \
  --name netrecon-probe \
  --network host \
  --restart unless-stopped \
  -e ENROLLMENT_TOKEN="your-token-from-dashboard" \
  netrecon/probe:latest
```

Λάβετε το token εγγραφής από **Τοποθεσίες → [Η Τοποθεσία σας] → Αισθητήρες → Προσθήκη Αισθητήρα → Docker**.

### Επιλογή Γ: Εικονική Μηχανή

1. Κατεβάστε το αρχείο OVA από τον Πίνακα Ελέγχου
2. Εισάγετε σε VMware, Proxmox ή Hyper-V
3. Ρυθμίστε τη VM με **bridged δικτύωση** (απαιτείται για σάρωση Layer 2)
4. Εκκινήστε τη VM — θα εμφανιστεί αυτόματα στον Πίνακα Ελέγχου σας

## Βήμα 4: Ξεκινήστε τη Σάρωση

Μόλις ο αισθητήρας είναι online:

1. Μεταβείτε στο **Τοποθεσίες → [Η Τοποθεσία σας] → Συσκευές**
2. Κάντε κλικ στο **Σάρωση Τώρα** ή περιμένετε την αυτόματη ανακάλυψη (εκτελείται κάθε 15 λεπτά)
3. Οι ανακαλυφθείσες συσκευές θα εμφανιστούν στην απογραφή συσκευών

## Βήμα 5: Εγκαταστήστε την Εφαρμογή Κινητού

Κατεβάστε το **NetRecon Scanner** από το Google Play Store για σάρωση δικτύου εν κινήσει:

- Σαρώστε οποιοδήποτε δίκτυο στο οποίο είναι συνδεδεμένο το τηλέφωνό σας
- Τα αποτελέσματα συγχρονίζονται αυτόματα στον cloud πίνακα ελέγχου σας
- Δείτε την [Επισκόπηση Scanner](../scanner/overview) για λεπτομέρειες

## Τι Ακολουθεί;

- **Αναπτύξτε agents** σε τερματικά για βαθύτερη ορατότητα → [Εγκατάσταση Agent](../agents/overview)
- **Ρυθμίστε ειδοποιήσεις** για νέες συσκευές, ευπάθειες ή διακοπές
- **Ρυθμίστε ενσωματώσεις** με τα υπάρχοντα εργαλεία σας (LDAP, SIEM, Jira, ServiceNow)
- **Προσκαλέστε την ομάδα σας** μέσω **Ρυθμίσεις → Διαχείριση Ομάδας**

## Cloud εναντίον Αυτο-φιλοξενίας

| Χαρακτηριστικό | Cloud | Αυτο-φιλοξενία |
|---|---|---|
| Διαχείριση διακομιστή | Διαχείριση από NetRecon | Εσείς διαχειρίζεστε |
| Τοποθεσία δεδομένων | NetRecon Cloud (ΕΕ) | Η δική σας υποδομή |
| Ενημερώσεις | Αυτόματες | Χειροκίνητες (docker pull) |
| Cloudflare Tunnel | Περιλαμβάνεται | Εσείς ρυθμίζετε |
| Τιμολόγηση | Συνδρομή | Κλειδί άδειας |

Χρειάζεστε αυτο-φιλοξενία αντ' αυτού; Δείτε τον [Οδηγό Εγκατάστασης](../self-hosting/installation).

Για βοήθεια, επικοινωνήστε με [support@netreconapp.com](mailto:support@netreconapp.com).
