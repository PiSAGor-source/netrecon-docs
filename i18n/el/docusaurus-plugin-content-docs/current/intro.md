# Ξεκινώντας με το NetRecon

Το NetRecon είναι μια πλατφόρμα ευφυΐας δικτύου σχεδιασμένη για MSP και ομάδες IT. Παρέχει αυτοματοποιημένη ανακάλυψη δικτύου, απογραφή συσκευών, σάρωση ευπαθειών, διαχείριση παραμετροποίησης και παρακολούθηση σε πραγματικό χρόνο — όλα προσβάσιμα μέσω ενός κεντρικού πίνακα ελέγχου, εφαρμογών κινητών και REST API.

## Επιλέξτε τον Τρόπο Ανάπτυξής σας

### Αυτο-φιλοξενία

Αναπτύξτε το NetRecon στη δική σας υποδομή χρησιμοποιώντας Docker Compose. Πλήρης έλεγχος των δεδομένων σας, χωρίς εξωτερικές εξαρτήσεις.

- [Απαιτήσεις Συστήματος](self-hosting/requirements)
- [Οδηγός Εγκατάστασης](self-hosting/installation)
- [Αναφορά Παραμετροποίησης](self-hosting/configuration)

**Ιδανικό για:** Οργανισμούς με αυστηρές απαιτήσεις κυριαρχίας δεδομένων, απομονωμένα δίκτυα ή υπάρχουσα υποδομή διακομιστών.

</div>

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Cloud (SaaS)

Ξεκινήστε αμέσως με το NetRecon Cloud. Δεν απαιτείται εγκατάσταση διακομιστή — απλά αναπτύξτε αισθητήρες και ξεκινήστε τη σάρωση.

- [Οδηγός Γρήγορης Εκκίνησης](cloud/quickstart)

**Ιδανικό για:** Ομάδες που θέλουν να ξεκινήσουν γρήγορα χωρίς να διαχειρίζονται υποδομή διακομιστών.

</div>

</div>

## Στοιχεία Πλατφόρμας

| Στοιχείο | Περιγραφή |
|---|---|
| **Πίνακας Ελέγχου** | Πίνακας ελέγχου μέσω web για όλες τις λειτουργίες του NetRecon |
| **NetRecon Scanner** | Εφαρμογή Android για σάρωση δικτύου εν κινήσει ([Μάθετε περισσότερα](scanner/overview)) |
| **Admin Connect** | Εφαρμογή διαχείρισης Android για απομακρυσμένη διαχείριση ([Μάθετε περισσότερα](admin-connect/overview)) |
| **Agents** | Ελαφροί agents για τερματικά Windows, macOS και Linux ([Εγκατάσταση](agents/overview)) |
| **Αισθητήρες** | Αισθητήρες δικτύου βασισμένοι σε hardware ή VM για συνεχή παρακολούθηση |
| **API** | RESTful API για αυτοματισμό και ενσωμάτωση ([Αναφορά API](api/overview)) |

## Χρειάζεστε Βοήθεια;

- Περιηγηθείτε στην τεκμηρίωση χρησιμοποιώντας την πλαϊνή μπάρα
- Ελέγξτε την [Αναφορά API](api/overview) για λεπτομέρειες ενσωμάτωσης
- Επικοινωνήστε με [support@netreconapp.com](mailto:support@netreconapp.com) για βοήθεια
