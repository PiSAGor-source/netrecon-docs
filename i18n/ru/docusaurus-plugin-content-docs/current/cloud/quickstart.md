# Быстрый старт с облаком

NetRecon Cloud — самый быстрый способ начать работу. Без настройки сервера, без Docker — просто зарегистрируйтесь, разверните зонд и начните обнаружение вашей сети.

## Шаг 1: Создайте учётную запись

1. Перейдите на [app.netreconapp.com](https://app.netreconapp.com) и нажмите **Зарегистрироваться**
2. Введите ваш email, название компании и пароль
3. Подтвердите адрес электронной почты
4. Войдите в панель управления NetRecon

## Шаг 2: Добавьте первую площадку

1. В панели управления перейдите в раздел **Площадки** на боковой панели
2. Нажмите **Добавить площадку**
3. Введите название и адрес площадки (например, «Главный офис — Москва»)
4. Сохраните площадку

## Шаг 3: Разверните зонд

Для каждой площадки необходим хотя бы один зонд для обнаружения и мониторинга сети.

### Вариант A: NetRecon OS (рекомендуется)

1. Перейдите в **Площадки → [Ваша площадка] → Зонды → Добавить зонд**
2. Выберите **NetRecon OS** и загрузите образ для вашего оборудования
3. Запишите образ на SD-карту или SSD с помощью [balenaEtcher](https://etcher.balena.io/)
4. Подключите зонд к вашей сети через Ethernet
5. Включите питание — зонд автоматически подключится к вашей облачной учётной записи через Cloudflare Tunnel

### Вариант B: Docker на существующем сервере

```bash
# Скачайте и запустите контейнер зонда
docker run -d \
  --name netrecon-probe \
  --network host \
  --restart unless-stopped \
  -e ENROLLMENT_TOKEN="your-token-from-dashboard" \
  netrecon/probe:latest
```

Получите токен регистрации в разделе **Площадки → [Ваша площадка] → Зонды → Добавить зонд → Docker**.

### Вариант C: Виртуальная машина

1. Загрузите файл OVA из панели управления
2. Импортируйте в VMware, Proxmox или Hyper-V
3. Настройте ВМ с **мостовой сетью** (необходимо для сканирования на уровне Layer 2)
4. Запустите ВМ — она автоматически появится в вашей панели управления

## Шаг 4: Начните сканирование

После того как зонд будет в сети:

1. Перейдите в **Площадки → [Ваша площадка] → Устройства**
2. Нажмите **Сканировать сейчас** или дождитесь автоматического обнаружения (выполняется каждые 15 минут)
3. Обнаруженные устройства появятся в инвентаре устройств

## Шаг 5: Установите мобильное приложение

Загрузите **NetRecon Scanner** из Google Play Store для сканирования сети на ходу:

- Сканируйте любую сеть, к которой подключён ваш телефон
- Результаты автоматически синхронизируются с вашей облачной панелью управления
- Подробнее см. в [Обзоре сканера](../scanner/overview)

## Что дальше?

- **Разверните агенты** на конечных устройствах для более глубокой видимости → [Установка агентов](../agents/overview)
- **Настройте оповещения** о новых устройствах, уязвимостях или простоях
- **Настройте интеграции** с вашими существующими инструментами (LDAP, SIEM, Jira, ServiceNow)
- **Пригласите свою команду** через **Настройки → Управление командой**

## Облако vs Локальное развёртывание

| Функция | Облако | Локальное развёртывание |
|---|---|---|
| Управление сервером | Управляется NetRecon | Управляете вы |
| Расположение данных | NetRecon Cloud (ЕС) | Ваша инфраструктура |
| Обновления | Автоматические | Ручные (docker pull) |
| Cloudflare Tunnel | Включён | Настраиваете вы |
| Ценообразование | Подписка | Лицензионный ключ |

Нужно локальное развёртывание? Смотрите [Руководство по установке](../self-hosting/installation).
