# Начало работы с NetRecon

NetRecon — это платформа сетевой аналитики, созданная для MSP и IT-команд. Она обеспечивает автоматическое обнаружение сетей, инвентаризацию устройств, сканирование уязвимостей, управление конфигурациями и мониторинг в реальном времени — всё это доступно через централизованную панель управления, мобильные приложения и REST API.

## Выберите способ развёртывания

### Локальное развёртывание

Разверните NetRecon на собственной инфраструктуре с помощью Docker Compose. Полный контроль над данными, без внешних зависимостей.

- [Системные требования](self-hosting/requirements)
- [Руководство по установке](self-hosting/installation)
- [Справочник по конфигурации](self-hosting/configuration)

**Лучше всего подходит для:** организаций со строгими требованиями к суверенитету данных, изолированных сетей или при наличии собственной серверной инфраструктуры.

</div>

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Облако (SaaS)

Начните работу мгновенно с NetRecon Cloud. Настройка сервера не требуется — просто разверните зонды и начните сканирование.

- [Краткое руководство](cloud/quickstart)

**Лучше всего подходит для:** команд, которые хотят быстро начать работу без управления серверной инфраструктурой.

</div>

## Компоненты платформы

| Компонент | Описание |
|---|---|
| **Dashboard** | Веб-панель управления для всех функций NetRecon |
| **NetRecon Scanner** | Android-приложение для сканирования сети на ходу ([Подробнее](scanner/overview)) |
| **Admin Connect** | Android-приложение для удалённого администрирования ([Подробнее](admin-connect/overview)) |
| **Агенты** | Лёгкие агенты для конечных устройств Windows, macOS и Linux ([Установка](agents/overview)) |
| **Зонды** | Аппаратные или виртуальные сетевые сенсоры для непрерывного мониторинга |
| **API** | RESTful API для автоматизации и интеграции ([Справочник API](api/overview)) |
