# Cloud क्विक स्टार्ट

NetRecon Cloud शुरू करने का सबसे तेज़ तरीका है। कोई सर्वर सेटअप नहीं, कोई Docker नहीं — बस साइन अप करें, प्रोब डिप्लॉय करें और अपने नेटवर्क की खोज शुरू करें।

## चरण 1: अपना अकाउंट बनाएँ

1. [app.netreconapp.com](https://app.netreconapp.com) पर जाएँ और **Sign Up** पर क्लिक करें
2. अपना ईमेल, कंपनी का नाम और पासवर्ड दर्ज करें
3. अपने ईमेल पते की पुष्टि करें
4. NetRecon Dashboard में लॉग इन करें

## चरण 2: अपनी पहली साइट जोड़ें

1. Dashboard में, साइडबार में **Sites** पर जाएँ
2. **Add Site** पर क्लिक करें
3. साइट के लिए नाम और पता दर्ज करें (उदा., "मुख्य कार्यालय — दिल्ली")
4. साइट सेव करें

## चरण 3: प्रोब डिप्लॉय करें

प्रत्येक साइट को नेटवर्क खोज और निगरानी के लिए कम से कम एक प्रोब की आवश्यकता होती है।

### विकल्प A: NetRecon OS (अनुशंसित)

1. **Sites → [आपकी साइट] → Probes → Add Probe** पर जाएँ
2. **NetRecon OS** चुनें और अपने हार्डवेयर के लिए इमेज डाउनलोड करें
3. [balenaEtcher](https://etcher.balena.io/) का उपयोग करके इमेज को SD कार्ड या SSD पर फ़्लैश करें
4. ईथरनेट के माध्यम से प्रोब को अपने नेटवर्क से कनेक्ट करें
5. पावर ऑन करें — प्रोब स्वचालित रूप से Cloudflare Tunnel के माध्यम से आपके क्लाउड अकाउंट से कनेक्ट हो जाएगा

### विकल्प B: मौजूदा सर्वर पर Docker

```bash
# प्रोब कंटेनर पुल और रन करें
docker run -d \
  --name netrecon-probe \
  --network host \
  --restart unless-stopped \
  -e ENROLLMENT_TOKEN="your-token-from-dashboard" \
  netrecon/probe:latest
```

एनरोलमेंट टोकन **Sites → [आपकी साइट] → Probes → Add Probe → Docker** से प्राप्त करें।

### विकल्प C: वर्चुअल मशीन

1. Dashboard से OVA फ़ाइल डाउनलोड करें
2. VMware, Proxmox, या Hyper-V में इम्पोर्ट करें
3. VM को **bridged networking** के साथ कॉन्फ़िगर करें (Layer 2 स्कैनिंग के लिए आवश्यक)
4. VM शुरू करें — यह स्वचालित रूप से आपके Dashboard में दिखाई देगा

## चरण 4: स्कैनिंग शुरू करें

प्रोब ऑनलाइन होने के बाद:

1. **Sites → [आपकी साइट] → Devices** पर जाएँ
2. **Scan Now** पर क्लिक करें या स्वचालित खोज की प्रतीक्षा करें (हर 15 मिनट में चलती है)
3. खोजे गए डिवाइस डिवाइस इन्वेंट्री में दिखाई देंगे

## चरण 5: मोबाइल ऐप इंस्टॉल करें

चलते-फिरते नेटवर्क स्कैनिंग के लिए Google Play Store से **NetRecon Scanner** डाउनलोड करें:

- अपने फ़ोन से कनेक्टेड किसी भी नेटवर्क को स्कैन करें
- परिणाम स्वचालित रूप से आपके क्लाउड डैशबोर्ड से सिंक होते हैं
- विवरण के लिए [Scanner अवलोकन](../scanner/overview) देखें

## आगे क्या?

- **एजेंट डिप्लॉय करें** — गहरी दृश्यता के लिए एंडपॉइंट्स पर → [एजेंट इंस्टॉलेशन](../agents/overview)
- **अलर्ट सेट अप करें** — नए डिवाइस, भेद्यताओं या डाउनटाइम के लिए
- **एकीकरण कॉन्फ़िगर करें** — अपने मौजूदा टूल्स (LDAP, SIEM, Jira, ServiceNow) के साथ
- **अपनी टीम को आमंत्रित करें** — **Settings → Team Management** के माध्यम से

## Cloud बनाम सेल्फ-होस्टेड

| सुविधा | Cloud | सेल्फ-होस्टेड |
|---|---|---|
| सर्वर प्रबंधन | NetRecon द्वारा प्रबंधित | आप प्रबंधित करते हैं |
| डेटा स्थान | NetRecon Cloud (EU) | आपका इन्फ्रास्ट्रक्चर |
| अपडेट | स्वचालित | मैनुअल (docker pull) |
| Cloudflare Tunnel | शामिल | आप कॉन्फ़िगर करते हैं |
| मूल्य निर्धारण | सदस्यता | लाइसेंस कुंजी |

सेल्फ-होस्टेड चाहिए? [इंस्टॉलेशन गाइड](../self-hosting/installation) देखें।

सहायता के लिए, [support@netreconapp.com](mailto:support@netreconapp.com) से संपर्क करें।
