# NetRecon के साथ शुरुआत करें

NetRecon एक नेटवर्क इंटेलिजेंस प्लेटफ़ॉर्म है जो MSP और IT टीमों के लिए बनाया गया है। यह स्वचालित नेटवर्क खोज, डिवाइस इन्वेंट्री, भेद्यता स्कैनिंग, कॉन्फ़िगरेशन प्रबंधन और रीयल-टाइम निगरानी प्रदान करता है — सब कुछ एक केंद्रीकृत डैशबोर्ड, मोबाइल ऐप्स और REST API के माध्यम से उपलब्ध है।

## अपना डिप्लॉयमेंट चुनें

### सेल्फ-होस्टेड

Docker Compose का उपयोग करके अपने स्वयं के इन्फ्रास्ट्रक्चर पर NetRecon डिप्लॉय करें। अपने डेटा पर पूर्ण नियंत्रण, कोई बाहरी निर्भरता नहीं।

- [सिस्टम आवश्यकताएँ](self-hosting/requirements)
- [इंस्टॉलेशन गाइड](self-hosting/installation)
- [कॉन्फ़िगरेशन संदर्भ](self-hosting/configuration)

**सबसे अच्छा:** सख्त डेटा संप्रभुता आवश्यकताओं, एयर-गैप्ड नेटवर्क, या मौजूदा सर्वर इन्फ्रास्ट्रक्चर वाले संगठनों के लिए।

</div>

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Cloud (SaaS)

NetRecon Cloud के साथ तुरंत शुरू करें। कोई सर्वर सेटअप आवश्यक नहीं — बस प्रोब डिप्लॉय करें और स्कैनिंग शुरू करें।

- [क्विक स्टार्ट गाइड](cloud/quickstart)

**सबसे अच्छा:** उन टीमों के लिए जो सर्वर इन्फ्रास्ट्रक्चर प्रबंधित किए बिना जल्दी शुरू करना चाहती हैं।

</div>

</div>

## प्लेटफ़ॉर्म घटक

| घटक | विवरण |
|---|---|
| **Dashboard** | सभी NetRecon सुविधाओं के लिए वेब-आधारित कंट्रोल पैनल |
| **NetRecon Scanner** | चलते-फिरते नेटवर्क स्कैनिंग के लिए Android ऐप ([और जानें](scanner/overview)) |
| **Admin Connect** | रिमोट प्रशासन के लिए Android प्रबंधन ऐप ([और जानें](admin-connect/overview)) |
| **Agents** | Windows, macOS और Linux एंडपॉइंट्स के लिए हल्के एजेंट ([इंस्टॉलेशन](agents/overview)) |
| **Probes** | निरंतर निगरानी के लिए हार्डवेयर या VM-आधारित नेटवर्क सेंसर |
| **API** | स्वचालन और एकीकरण के लिए RESTful API ([API संदर्भ](api/overview)) |

## सहायता चाहिए?

- साइडबार का उपयोग करके दस्तावेज़ीकरण ब्राउज़ करें
- एकीकरण विवरण के लिए [API संदर्भ](api/overview) देखें
- सहायता के लिए [support@netreconapp.com](mailto:support@netreconapp.com) से संपर्क करें
