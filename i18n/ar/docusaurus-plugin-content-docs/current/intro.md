# البدء مع NetRecon

NetRecon هي منصة ذكاء شبكات مصممة لمزودي الخدمات المُدارة (MSPs) وفرق تكنولوجيا المعلومات. توفر اكتشافًا تلقائيًا للشبكة، وجردًا للأجهزة، وفحصًا للثغرات الأمنية، وإدارة التكوين، والمراقبة في الوقت الفعلي — كل ذلك متاح من خلال لوحة تحكم مركزية، وتطبيقات الهاتف المحمول، وواجهة REST API.

## مكونات المنصة

| المكون | الوصف |
|---|---|
| **لوحة التحكم** | لوحة تحكم قائمة على الويب لجميع ميزات NetRecon |
| **NetRecon Scanner** | تطبيق Android لفحص الشبكات أثناء التنقل ([اعرف المزيد](scanner/overview)) |
| **Admin Connect** | تطبيق Android للإدارة عن بُعد ([اعرف المزيد](admin-connect/overview)) |
| **الوكلاء** | وكلاء خفيفون لأجهزة Windows وmacOS وLinux ([التثبيت](agents/overview)) |
| **المجسات** | مستشعرات شبكية قائمة على الأجهزة أو الأجهزة الافتراضية للمراقبة المستمرة |
| **API** | واجهة RESTful API للأتمتة والتكامل ([مرجع API](api/overview)) |
