# Kom i gang med NetRecon

NetRecon er en nettverksetterretningsplattform bygget for MSP-er og IT-team. Den tilbyr automatisert nettverksoppdagelse, enhetsoversikt, sarbarhetsskanning, konfigurasjonsadministrasjon og sanntidsovervaking -- alt tilgjengelig gjennom et sentralisert dashbord, mobilapper og REST API.

### Selvhostet

Distribuer NetRecon pa din egen infrastruktur med Docker Compose. Full kontroll over dine data, uten eksterne avhengigheter.

- [Systemkrav](self-hosting/requirements)
- [Installasjonsguide](self-hosting/installation)
- [Konfigurasjonsreferanse](self-hosting/configuration)

**Best egnet for:** Organisasjoner med strenge krav til datasuverenitet, isolerte nettverk eller eksisterende serverinfrastruktur.

<div className="col col--6" style={{marginBottom: '1rem'}}>

### Sky (SaaS)

Kom i gang umiddelbart med NetRecon Cloud. Ingen serveroppsett nodvendig -- bare distribuer prober og begynn a skanne.

- [Hurtigstartguide](cloud/quickstart)

**Best egnet for:** Team som onsker a komme raskt i gang uten a administrere serverinfrastruktur.

## Plattformkomponenter

| Komponent | Beskrivelse |
|---|---|
| **Dashbord** | Nettbasert kontrollpanel for alle NetRecon-funksjoner |
| **NetRecon Scanner** | Android-app for nettverksskanning pa farten ([Les mer](scanner/overview)) |
| **Admin Connect** | Android-administrasjonsapp for fjernkontroll ([Les mer](admin-connect/overview)) |
| **Agenter** | Lette agenter for Windows, macOS og Linux-endepunkter ([Installasjon](agents/overview)) |
| **Prober** | Maskinvare- eller VM-baserte nettverkssensorer for kontinuerlig overvaking |
| **API** | RESTful API for automatisering og integrasjon ([API-referanse](api/overview)) |
