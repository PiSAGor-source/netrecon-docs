# Primii pași cu NetRecon

NetRecon este o platformă de informații despre rețea creată pentru MSP-uri și echipe IT. Oferă descoperire automată a rețelei, inventar de dispozitive, scanare de vulnerabilități, managementul configurațiilor și monitorizare în timp real — toate accesibile printr-un tablou de bord centralizat, aplicații mobile și API REST.

## Alegeți tipul de implementare

### Auto-găzduit

Implementați NetRecon pe propria infrastructură folosind Docker Compose. Control complet asupra datelor, fără dependențe externe.

- [Cerințe de sistem](self-hosting/requirements)
- [Ghid de instalare](self-hosting/installation)
- [Referință de configurare](self-hosting/configuration)

**Cel mai potrivit pentru:** Organizații cu cerințe stricte de suveranitate a datelor, rețele izolate sau infrastructură server existentă.

### Cloud (SaaS)

Începeți instantaneu cu NetRecon Cloud. Nu este necesară configurarea serverului — doar implementați sonde și începeți scanarea.

- [Ghid de pornire rapidă](cloud/quickstart)

**Cel mai potrivit pentru:** Echipe care doresc să pornească rapid fără a gestiona infrastructura server.

## Componentele platformei

| Componentă | Descriere |
|---|---|
| **Tablou de bord** | Panou de control web pentru toate funcționalitățile NetRecon |
| **NetRecon Scanner** | Aplicație Android pentru scanarea rețelei din mers ([Aflați mai multe](scanner/overview)) |
| **Admin Connect** | Aplicație Android de administrare pentru management la distanță ([Aflați mai multe](admin-connect/overview)) |
| **Agenți** | Agenți ușori pentru punctele terminale Windows, macOS și Linux ([Instalare](agents/overview)) |
| **Sonde** | Senzori de rețea bazați pe hardware sau VM pentru monitorizare continuă |
| **API** | API RESTful pentru automatizare și integrare ([Referință API](api/overview)) |
