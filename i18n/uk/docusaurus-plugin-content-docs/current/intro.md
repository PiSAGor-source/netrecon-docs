# Початок роботи з NetRecon

NetRecon — це платформа мережевої розвідки, створена для MSP та IT-команд. Вона забезпечує автоматичне виявлення мережі, інвентаризацію пристроїв, сканування вразливостей, управління конфігураціями та моніторинг у реальному часі — все це доступно через централізовану панель керування, мобільні додатки та REST API.

Розгорніть NetRecon на власній інфраструктурі за допомогою Docker Compose. Повний контроль над вашими даними, без зовнішніх залежностей.

**Найкраще підходить для:** Організацій із суворими вимогами до суверенітету даних, ізольованих мереж або наявної серверної інфраструктури.

### Хмара (SaaS)

Почніть миттєво з NetRecon Cloud. Не потрібно налаштовувати сервер — просто розгорніть зонди та починайте сканування.

- [Швидкий старт](cloud/quickstart)

**Найкраще підходить для:** Команд, які хочуть швидко розпочати роботу без управління серверною інфраструктурою.

## Компоненти платформи

| Компонент | Опис |
|---|---|
| **Панель керування** | Веб-панель керування для всіх функцій NetRecon |
| **NetRecon Scanner** | Android-додаток для сканування мережі на ходу ([Детальніше](scanner/overview)) |
| **Admin Connect** | Android-додаток для віддаленого адміністрування ([Детальніше](admin-connect/overview)) |
| **Агенти** | Легковагі агенти для Windows, macOS та Linux ([Встановлення](agents/overview)) |
| **Зонди** | Апаратні або віртуальні мережеві сенсори для безперервного моніторингу |
| **API** | RESTful API для автоматизації та інтеграції ([Довідник API](api/overview)) |
