# Швидкий старт з хмарою

NetRecon Cloud — це найшвидший спосіб розпочати роботу. Без налаштування сервера, без Docker — просто зареєструйтесь, розгорніть зонд і починайте виявляти свою мережу.

## Крок 1: Створіть обліковий запис

1. Перейдіть на [app.netreconapp.com](https://app.netreconapp.com) та натисніть **Sign Up**
2. Введіть електронну пошту, назву компанії та пароль
3. Підтвердіть вашу електронну адресу
4. Увійдіть до панелі керування NetRecon

## Крок 2: Додайте перший сайт

1. На панелі керування перейдіть до **Sites** на бічній панелі
2. Натисніть **Add Site**
3. Введіть назву та адресу сайту (наприклад, "Головний офіс — Київ")
4. Збережіть сайт

## Крок 3: Розгорніть зонд

Кожен сайт потребує принаймні один зонд для виявлення мережі та моніторингу.

### Варіант A: NetRecon OS (Рекомендовано)

1. Перейдіть до **Sites → [Ваш сайт] → Probes → Add Probe**
2. Оберіть **NetRecon OS** та завантажте образ для вашого обладнання
3. Запишіть образ на SD-карту або SSD за допомогою [balenaEtcher](https://etcher.balena.io/)
4. Підключіть зонд до мережі через Ethernet
5. Увімкніть живлення — зонд автоматично підключиться до вашого хмарного облікового запису через Cloudflare Tunnel

### Варіант B: Docker на існуючому сервері

```bash
# Завантажте та запустіть контейнер зонда
docker run -d \
  --name netrecon-probe \
  --network host \
  --restart unless-stopped \
  -e ENROLLMENT_TOKEN="ваш-токен-з-панелі-керування" \
  netrecon/probe:latest
```

Отримайте токен реєстрації у **Sites → [Ваш сайт] → Probes → Add Probe → Docker**.

### Варіант C: Віртуальна машина

1. Завантажте файл OVA з панелі керування
2. Імпортуйте у VMware, Proxmox або Hyper-V
3. Налаштуйте ВМ з **мостовою мережею** (необхідно для сканування рівня 2)
4. Запустіть ВМ — вона автоматично з'явиться у вашій панелі керування

## Крок 4: Почніть сканування

Коли зонд онлайн:

1. Перейдіть до **Sites → [Ваш сайт] → Devices**
2. Натисніть **Scan Now** або дочекайтесь автоматичного виявлення (запускається кожні 15 хвилин)
3. Виявлені пристрої з'являться в інвентарі пристроїв

## Крок 5: Встановіть мобільний додаток

Завантажте **NetRecon Scanner** з Google Play Store для сканування мережі на ходу:

- Скануйте будь-яку мережу, до якої підключений ваш телефон
- Результати автоматично синхронізуються з хмарною панеллю керування
- Дивіться [Огляд Scanner](../scanner/overview) для деталей

## Що далі?

- **Розгорніть агенти** на кінцевих точках для глибшої видимості → [Встановлення агентів](../agents/overview)
- **Налаштуйте сповіщення** для нових пристроїв, вразливостей або простоїв
- **Налаштуйте інтеграції** з вашими існуючими інструментами (LDAP, SIEM, Jira, ServiceNow)
- **Запросіть свою команду** через **Settings → Team Management**

## Хмара vs Власний сервер

| Функція | Хмара | Власний сервер |
|---|---|---|
| Управління сервером | Керується NetRecon | Ви керуєте |
| Розташування даних | NetRecon Cloud (ЄС) | Ваша інфраструктура |
| Оновлення | Автоматичні | Ручні (docker pull) |
| Cloudflare Tunnel | Включено | Ви налаштовуєте |
| Тарифікація | Підписка | Ліцензійний ключ |

Потрібен власний сервер? Дивіться [Інструкцію з встановлення](../self-hosting/installation).
