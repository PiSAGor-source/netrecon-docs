# Aan de slag met NetRecon

NetRecon is een netwerkintelligentieplatform ontwikkeld voor MSP's en IT-teams. Het biedt geautomatiseerde netwerkdetectie, apparaatinventarisatie, kwetsbaarheidsscanning, configuratiebeheer en real-time monitoring — allemaal toegankelijk via een centraal dashboard, mobiele apps en een REST API.

### Self-hosted

Implementeer NetRecon op uw eigen infrastructuur met Docker Compose. Volledige controle over uw gegevens, geen externe afhankelijkheden.

- [Systeemvereisten](self-hosting/requirements)
- [Installatiehandleiding](self-hosting/installation)
- [Configuratiereferentie](self-hosting/configuration)

**Aanbevolen voor:** Organisaties met strikte eisen voor gegevenssoevereiniteit, air-gapped netwerken of bestaande serverinfrastructuur.

### Cloud (SaaS)

Ga direct aan de slag met NetRecon Cloud. Geen serverinstallatie nodig — implementeer probes en begin met scannen.

- [Snelstartgids](cloud/quickstart)

**Aanbevolen voor:** Teams die snel aan de slag willen zonder serverinfrastructuur te beheren.

## Platformonderdelen

| Onderdeel | Beschrijving |
|---|---|
| **Dashboard** | Webgebaseerd bedieningspaneel voor alle NetRecon-functies |
| **NetRecon Scanner** | Android-app voor netwerkscannen onderweg ([Meer informatie](scanner/overview)) |
| **Admin Connect** | Android-beheerapp voor extern beheer ([Meer informatie](admin-connect/overview)) |
| **Agents** | Lichtgewicht agents voor Windows-, macOS- en Linux-eindpunten ([Installatie](agents/overview)) |
| **Probes** | Hardware- of VM-gebaseerde netwerksensoren voor continue monitoring |
| **API** | RESTful API voor automatisering en integratie ([API-referentie](api/overview)) |
